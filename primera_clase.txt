git init::::::: Crea un repositorio
git status:::::Ver archivos::: Rojo::No esta "TRakeado" no esta listo para subir
				Verde:: Esta trakeado, listo para subir
git add:::::::: Pushea archivo. Trakea el archivo. Prepara el archivo para un "commit". pueden ir muchos archivos
git commit -m "mensaje"::::: crea un punto en el tiempo y sube los archivos trakeados
git log:::::::: Ver el historial de commits
